# SpaceShmup

**SpaceShmup** is a single-player arcade-style space shooter built with **Unity**.  
The game is inspired by classic shoot-em-up (shmup) games, featuring fast-paced combat, enemy waves, power-ups, shields, and a scrolling space background.

This project was developed as a learning and portfolio project to demonstrate core game development concepts such as object-oriented design, collision handling, UI systems, and gameplay logic.

---

## Gameplay Overview

- Control a hero spaceship
- Shoot incoming enemy ships
- Dodge enemy movement patterns
- Break enemy shields and reduce health
- Collect power-ups to change weapons
- Score points by defeating enemies
- Survive as long as possible

---

## Features

- Player-controlled spaceship with multiple weapons
- Enemy waves with different movement patterns
- Shield and health systems
- Weapon upgrades via power-ups
- Scoring and lives system
- Parallax scrolling background
- Visual hit feedback for damage
- Game UI with score and game-over state
